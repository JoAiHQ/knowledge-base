# Templates

This directory contains templates for creating knowledge base articles.

## Article Template

Use `article-template.md` as a starting point for new articles. It includes:

- Required front matter fields
- Optional metadata fields
- Standard article structure
- Best practices for formatting

## Usage

1. Copy `article-template.md` to the appropriate category directory
2. Fill in the front matter with article-specific information
3. Write your content following the structure
4. Add images to the corresponding `images/{category}/{slug}/` directory
